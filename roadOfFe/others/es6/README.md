### let & const  
var有变量提升，let和const没有；  
let和const有块级作用域，var没有；  

暂时性死区的本质：只要一进入当前作用域，所要使用的变量就已经存在了，但是不可获取，只有等声明变量的那一行代码出现才可以获取和使用该变量。
